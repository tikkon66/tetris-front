  Frontend онлайн-игры «Тетрис», разработанный на React + TypeScript + Vite.  

Реализована полноценная игровая логика тетриса:  
• движение и вращение фигур;  
• система коллизий;  
• очистка линий;  
• подсчёт очков;  
• увеличение сложности и скорости игры;    
• отображение следующей фигуры;  
• Game Over и перезапуск игры.  
  
Функционал приложения:  
• регистрация и авторизация пользователей;  
• сохранение рекордов через backend API;  
• вывод глобальной таблицы лидеров;  
• отображение личного рекорда игрока;  
• синхронизация данных с сервером;  
• адаптивный интерфейс и современный UI.  

Технологии:    
• React  
• TypeScript  
• Vite  
• CSS / Responsive Design  
• Fetch API / Axios  
• JWT Authentication  
• REST API integration  

Проект построен с использованием компонентной архитектуры, React Hooks и клиент-серверного взаимодействия. Реализована работа с авторизацией, защищёнными запросами и хранением пользовательских данных.
